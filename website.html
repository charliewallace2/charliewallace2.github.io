AI-Integrated Website

August 2025 - December 2025


Description:

I was tasked to create a website using HTML, JavaScript, and Node.js that utilized ChatGPT to answer a curated prompt. My project was about generating creative writing prompts, where the user was asked their writing level, preferred genre, subgenre, POV, and how many prompts they wanted generated. I then used JavaScript to send data to the Node.js server, where the data was checked using express-validator, placed into a prompt using the body’s formData, sent to ChatGPT, and then printed out the response from the AI onto the website itself for the user to see.
