AR Platformer Game

November 2024 - December 2024


Description:

This is a group project in which we were to create a platformer game that uses an AR scanner. For this project, we used the scanner to scan cards from a deck of playing cards, add the card to a counter on the player’s screen, and then allow the player to use various “boosts” while completing the game if they pressed a designated key. A large portion of this project was focused on coding in Unity with C# for the game and using and creating compatibility with Vuforia for the AR scanner. The artwork and assets were taken from our own individual platformer games (previously mentioned). My group tried to use GitHub to make working on the project together more efficient, but there was an issue with the .gitignore and Vuforia which made the project impossible to update on GitHub, so we ended up putting the entire project on my laptop and spending almost five consecutive hours finalizing the game.
